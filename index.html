Programming

I’ve been writing code for over a decade, and over that time many languages have had the dubious distinction of being my “main” language. I cut my teeth on TI-BASIC in high school and (more formally) on Java in undergrad, wrote a ton of C++ and Fortran90 for my graduate thesis, hacked Perl and Bash scripts together for Wall Street until I fell for Python. Sys admining a MongoDB cluster turned me on to Javascript. From there, I found AngularJS and Node which opened me up to a brand new world. Currently, I’m writing hybrid mobile apps using the Ionic Framework and Cordova. It’s been a humbling experience.

Python is still my area of expertise, but I’m learning more every day about modern client-side web technologies as well as Android and iOS integration. I used to be fairly dogmatic about object-oriented design (as is probably anyone that started with Java), but have lately been writing procedurally. I haven’t decided whether this produces better Javascript or if it’s just part of coping with the learning curve.

I like to build things

I've built mobile apps for both Android and iOS, written internal tools for a top hft firm, released a pde solver while in grad. school, and designed a thrust sensor for a rocket. I've dabbled in home brewing, constructed a trebuchet, and enjoy fixing a 1999 Jeep Cherokee.

Project management makes sense to me. If you have something you'd like to build, and you think I could help you, then please reach out.
