Programming

I’ve been writing code for over a decade, and over that time many languages have had the dubious distinction of being my “main” language. I cut my teeth on TI-BASIC in high school and (more formally) on Java in undergrad, wrote a ton of C++ and Fortran90 for my graduate thesis, hacked Perl and Bash scripts together for Wall Street and eventually fell for Python. Sys admining a MongoDB cluster turned me on to Javascript and, currently, I’m writing hybrid mobile apps using Ionic, AngularJS, and Cordova. It’s been a humbling experience.

Python is still my area of expertise, but I’m learning more every day about the modern client-side web as well as Android and iOS integration. I used to be fairly dogmatic about object-oriented design (as is probably anyone that started with Java), now I favor a more balanced approach.
